## 🚀 طريقة التثبيت على Android

### الخطوة 1: تثبيت الأدوات

#### على Windows:
1. حمّل **Node.js** من: https://nodejs.org/ (الإصدار LTS)
2. عند التثبيت، تأكد من اختيار "Add to PATH"

#### على Linux:
```bash
sudo apt update
sudo apt install nodejs npm
```

### الخطوة 2: بناء التطبيق
افتح Terminal في هذا المجلد وشغل:

```bash
# Linux/Mac
chmod +x BUILD-APK.sh
./BUILD-APK.sh

# Windows (PowerShell)
.\BUILD-APK.ps1
```

أو يدوياً:
```bash
npm install
npm run build
npx cap sync android
cd android
chmod +x gradlew
./gradlew assembleDebug
```

### الخطوة 3: نقل APK للهاتف
- ملف APK موجود في: `android/app/build/outputs/apk/debug/app-debug.apk`
- انقله لهاتفك وثبته
- قد تحتاج تفعيل "مصادر غير معروفة" في إعدادات الهاتف
